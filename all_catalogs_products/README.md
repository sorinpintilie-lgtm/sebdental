# Multi-catalog extraction output

This folder contains product extraction results generated from all PDF catalogs in the project root.

## Main outputs

- `all_products.json` – combined product records from all catalogs
- `all_products.csv` – combined tabular export
- `summary.json` – per-catalog extracted product counts

## Per-catalog outputs

Each catalog has its own subfolder containing:

- `products.json`
- `products.csv`
- `images/` (extracted catalog images)
